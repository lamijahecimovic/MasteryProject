# Smart Document Processing System

## Live Demo
[https://document-processor.web.app](https://document-processor.web.app)

## Tech Stack
- **Frontend:** Flutter Web
- **Backend:** ASP.NET Core Web API
- **Database:** SQLite
- **AI/OCR:** Integration of Tesseract libraries for local text extraction from image documents

## Local Setup

### Backend
cd DocumentProcessor 
dotnet run

### Frontend
cd document_processor_ui 
flutter run -d chrome

## Overview
The system parses PDF, CSV, TXT, and image documents, extracting structured data while automatically validating calculations, dates, and duplicates.

## AI Tools Used
- **Claude Chat** for development assistance

## Future Improvements
